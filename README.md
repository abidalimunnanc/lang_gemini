# lang_gemini Agent

Simple Langchain Agent to try as command line tool

Github token 
setup Datastax database  get api key 
get gemni vertex ai api key

put it in .env file 

``` 
GITHUB_TOKEN = ""
ASTRA_DB_API_ENDPOINT = ""
ASTRA_DB_APPLICATION_TOKEN = ""
ASTRA_DB_KEYSPACE = ""
GOOGLE_API_KEY = ""
```


``` pip install -r requirements.txt```

``` python main.py ```
